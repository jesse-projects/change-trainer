# Sound Files

Add your downloaded MP3 sound files here:

- `key-tap.mp3` - Numpad key press sound
- `coin-tap.mp3` - Coin selection sound
- `bill-tap.mp3` - Bill selection sound
- `success.mp3` - Correct answer "cha-ching" sound
- `error.mp3` - Incorrect answer buzzer sound
- `ui-tap.mp3` - Menu/button tap sound

These should be the MP3 files you downloaded from ZapSplat.

Once added, the app will automatically load them instead of using the synthetic Web Audio API sounds.
